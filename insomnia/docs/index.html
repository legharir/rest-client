send

Sends a request with the specificied parameters as configuration

send(method: string, url: string, params: Object, headerWithoutAuth: Object, auth: Object, data: Object)
Parameters
method (string) any one of 'GET' or 'PUT'
url (string) the base request URL
params (Object) the query parameters specified as key value pairs
headerWithoutAuth (Object) the header of the request, without authorization
auth (Object) contains the user, pass, and token keys and values
data (Object) contains the response body
